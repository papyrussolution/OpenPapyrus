;
; Параметр для обратной совместимости с версиями
; более ранними, чем 4.0.5.
; При новом методе общая скидка точнее распределяется по товарным строкам.
;
; Новый алгоритм состоит в следующем:
; 1. Рассчитывается общая сумма по всем строкам - A
; 2. Общая скидка на документ - D
; 3. Попутно находим строку, имеющую минимальное количество товара q[i] при
;    максимальной цене p[i]. Запоминаем ее индекс last_position.
; 4. Пробегаем все строки за исключением [last_position] и расчитываем скидку
;    на каждую из этих строк по индуктивной формуле
;    d[i] = round(p[i] * (D - s[i]) / (A - a[i]), 2)
;    где s[i] = sum{j:0..i-1}( d[j] )
;        a[i] = sum{j:0..i-1}( p[j] * q[j] )
; 5. Наконец, скидку по строке last_position рассчитываем по формуле
;    d[last_position] = round(s[i] / q[last_position], 2)
;
; Старый алгорит был следующим:
; 1. Рассчитывается общая сумма по всем строкам - A
; 2. Общая скидка на документ - D

;    d[i] = round(p[i] * D / A, 2)
;

; Параметр для обратной совместимости с версиями,
